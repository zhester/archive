Data Plotter

Most math packages and charting programs these days can generate 3D charts, but the capabilities and control over those charts is generally very limited. This script is my very first attempt at building a plugin for Blender to facilitate a mesh generation system to render 3D charts.

An Example

Let's say I have some data for some important purpose. I've exported it from my preferred data/math package to a neutral CSV format:

The Blender plugin loads the CSV data directly from the file, and charts this information as a field of bars (currently, the only chart type available). Each series sits in its own row down the X axis. The series are separated from each other down the Y axis.

In the below image, the first series (the first column in the CSV file) is positioned closer to the camera. The second series is further from the camera. The values are displayed in order from left to right (top to bottom in the CSV file).

Future Development

I am trying to add support for pie charts and other chart types (just about anything that looks good illustrated in three dimensions). Once I have broadened the capabilities, I'm going to revise the UI, and add as much customization to the mesh generation options as I deem necessary.

The Code

The script is written as a Blender plugin script, but it has not been tested as a plugin yet (I just have the script loaded into the script editor in Blender). I will post more updates here as I can.
